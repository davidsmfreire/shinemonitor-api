# ShineMonitor

<img src="https://raw.githubusercontent.com/davidsmfreire/shinemonitor-api/main/shinemonitor.png" alt="ShineMonitor" width="120" align="right" />

Home Assistant integration for inverters that report to
`api.shinemonitor.com` (WatchPower, SolarPower, and similar vendor
apps).

## What you get

- One device per inverter on your account.
- Live sensors: grid voltage / frequency, AC output voltage / power /
  load, PV input voltage / power, battery voltage / capacity / charge
  current, etc.

## Setup

Add the integration via _Settings → Devices & Services → Add
Integration → ShineMonitor_ and enter the username and password you
use in the WatchPower / SolarPower app.
